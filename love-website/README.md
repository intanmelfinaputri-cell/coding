# 💕 Website Cinta Untuk Kiyo Tercinta 💕

## 🌸 Deskripsi
Website spesial yang dibuat dengan penuh cinta untuk Kiyo tersayang. Website ini berisi pesan-pesan menyentuh hati, kata-kata gombal romantis, dan berbagai game interaktif yang menyenangkan!

## 📅 Tanggal Spesial
**Pertemuan Pertama: 4 Maret** 🌸

## ✨ Fitur-Fitur

### 1. 💌 Pesan Cinta
- Tiga pesan menyentuh hati dengan animasi fade-in
- Kata-kata romantis yang penuh makna
- Gombalan yang lembut namun sincere

### 2. 🎮 Game Interaktif

#### 📝 Teka-Teki Silang
- Kisi silang untuk menemukan nama "KIYO"
- Petunjuk yang romantis
- Tombol reveal untuk melihat jawaban

#### 🔍 Tebak-Tebakan Romantis
- 5 riddle tentang cinta dan hubungan kalian
- Pilihan ganda dengan feedback
- Navigasi antar soal yang mudah

#### 🧩 Memory Game
- Cocokkan pasangan kartu dengan emoji cinta
- Grid 4x4 dengan berbagai emoji romantis
- Pesan kemenangan khusus

#### 💝 Quiz Tentang Kita
- 5 pertanyaan tentang hubungan dan perasaan
- Jawaban yang menunjukkan perasaan mendalam
- Feedback positif untuk setiap jawaban

### 3. 💕 Pertanyaan Akhir
**"Bibi mau jadi pacar mas ga?"**

- Tombol YES untuk menerima
- Tombol NO yang bergerak ketika di-hover/klik
- Setelah 5x mencoba klik NO, otomatis berhasil
- Animasi confetti jatuh ketika YES dipilih
- Pesan sukses yang romantis

### Efek Visual
- ❤️ Floating hearts yang bergerak mengambang di background
- Animasi smooth untuk semua elemen
- Responsive design untuk berbagai ukuran layar

## 📱 Responsif
- Mobile-friendly design
- Optimized untuk tablet dan desktop
- Semua game berfungsi dengan baik di semua perangkat

## 🚀 Cara Menggunakan

1. Buka file `index.html` di browser
2. Nikmati pesan-pesan cinta di bagian atas
3. Mainkan semua game yang tersedia
4. Jawab pertanyaan akhir dengan "YES"!
5. Nikmati animasi confetti dan pesan sukses 🎉
